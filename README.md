# Manashift

**Manashift** は、テキストをAIが「小説・漫画・フラッシュカード・動画スクリプト・詩」などの多様なフォーマットに変換し、コミュニティで共有できる学習コンテンツ変換プラットフォームです。学校・教育機関での導入を想定して設計されています。

## コンセプト

「学び（まなび）」＋「シフト（変換）」。同じ学習素材でも、人によって最も頭に入りやすいフォーマットは異なります。Manashiftは、ひとつのテキストを複数の認知スタイルに合わせた形式に変換することで、すべての学習者が自分に合った方法で学べる環境を提供します。

## 主な機能

| 機能 | 説明 |
| :--- | :--- |
| **コンテンツ生成** | テキストを Novel / Manga / Flashcard / Video Script / Poem に変換 |
| **AIカバー画像生成** | 各コンテンツに合わせたカバー画像を自動生成 |
| **コミュニティ共有** | 生成したコンテンツを公開・非公開で共有 |
| **ディスカバリー** | キーワード・フォーマットでコンテンツを検索・発見 |
| **いいね・コメント・ブックマーク** | コミュニティインタラクション機能 |
| **プロフィールページ** | ユーザーの作品一覧・自己紹介 |

## 技術スタック

| レイヤー | 技術 |
| :--- | :--- |
| フロントエンド | React 19 + TypeScript + Tailwind CSS 4 |
| バックエンド | Express + tRPC 11 |
| データベース | MySQL (Drizzle ORM) |
| AI | Built-in LLM API (Gemini) + Image Generation API |
| 認証 | Manus OAuth |
