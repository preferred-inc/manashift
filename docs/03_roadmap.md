# Manashift — ロードマップ・ビジョン

**バージョン:** 1.0  
**作成日:** 2026-03-02

---

## 1. ビジョン

**「すべての学習者が、自分に最適な形で知識を消化できる世界を作る。」**

Manashiftは、学習スタイルの多様性を技術で解決するプラットフォームである。テキストを読むのが苦手な人、音楽で覚えたい人、物語として体験したい人——それぞれに「刺さる形」でコンテンツを届けることで、学習の機会均等を実現する。

特に、ディスレクシア（読み書き困難）や学習障害を持つ学習者にとって、テキスト以外の形式での学習機会は非常に重要である。Manashiftはこうした多様なニーズに応えるインクルーシブな学習プラットフォームを目指す。

### Phase 2: コミュニティ深化（v1.1）

ユーザーエンゲージメントと発見体験の強化を目的とする。

| 機能 | 概要 |
| :--- | :--- |
| **タグ検索・絞り込み** | `#日本史` `#TOEIC` などのタグでコンテンツを分類・検索 |
| **週間ランキング** | いいね数・閲覧数によるトレンドランキング |
| **リミックス機能** | 他者のコンテンツを別フォーマットに変換（例：NovelをFlashcardに） |
| **フォロー・フィード** | フォローしたユーザーの新着コンテンツをタイムラインで表示 |

**リミックス機能**は本プラットフォームの最大の差別化要素である。「誰かが作った暗記小説を、自分はフラッシュカードで学びたい」というニーズに応え、コンテンツが複数の形式で派生・流通するエコシステムを生み出す。TikTokのデュエット機能の学習版と位置づける。

### Phase 3: インプット拡張（v1.2）

入力ソースを拡充し、より実用的な学習シーンに対応する。

| 機能 | 概要 |
| :--- | :--- |
| **画像インプット** | 教科書・ノートの写真をAIがOCR解析して変換 |
| **PDFインプット** | 参考書・資料PDFをアップロードして変換 |
| **URLインプット** | WebページのURLを指定して内容を変換 |

### Phase 4: フォーマット拡張（v2.0）

新しい変換モードを追加し、より多様な学習スタイルに対応する。

| モード | 概要 |
| :--- | :--- |
| **🎮 Visual Novel** | キャラクターが登場するノベルゲーム形式。ブラウザ上でプレイ可能 |
| **🗺️ Mind Map** | 内容を階層構造のインタラクティブマインドマップとして可視化 |
| **🎙️ Podcast** | 二人のホストが議論する対話形式の音声コンテンツ |
| **🎵 Song（暗記ソング）** | Mureka AI API連携による楽曲生成（外部API依存） |

### Phase 5: 教育機関展開（v3.0）

学校・塾・予備校への本格導入を目的とした機能拡充。

| 機能 | 概要 |

## 3. 差別化戦略

### 3.1. 競合に対するポジショニング

Manashiftは既存サービスのどれとも異なるポジションを占める。

- **Utamemo・Suno AI に対して:** 音楽生成に特化せず、5種類以上のフォーマットを提供。コミュニティ機能が充実。
- **Anki・Quizlet に対して:** フラッシュカード以外の多様な形式を提供。AI生成により素材作成の手間を大幅削減。
- **Notion AI・ChatGPT に対して:** 学習コンテキスト（タグ・科目）を保持し、コミュニティで共有・発見できる。

### 3.2. 学校導入のハードル低減

教育機関への導入において、以下の点を重視する。

**コスト面:** 無料プランで基本機能を体験でき、学校単位での一括契約（Schoolプラン）でコストを抑えられる設計とする。

**安全性:** 生成コンテンツのモデレーション機能、不適切コンテンツの報告機能を整備する。

**プライバシー:** 生徒の学習データを適切に管理し、GDPRおよび日本の個人情報保護法に準拠した設計とする。

**操作性:** 教員・生徒ともに直感的に操作できるUIを維持し、ICTリテラシーが高くないユーザーでも使えるようにする。
