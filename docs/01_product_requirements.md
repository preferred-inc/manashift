## 1. プロダクト概要

### 1.1. コンセプト

> **「あなたの学びを、あなたの形に。」**

Manashift（まなシフト）は、テキストをAIが複数のメディア形式に自動変換し、コミュニティで共有・発見できる**マルチフォーマット学習コンテンツ変換プラットフォーム**である。「学び（まなび）」と「シフト（変換・転換）」を組み合わせた造語であり、学習スタイルの多様性に応える設計思想を名称に込めている。

人間の記憶・学習スタイルは多様である。テキストを読むより歌で覚えたい人、物語として体験したい人、カードで反復したい人——Manashiftは同一の素材から複数の形式を生成することで、すべての学習者が「自分に最適な形」で知識を消化できる環境を提供する。

### 1.2. 背景と課題

既存の学習支援ツールは、「テキスト→フラッシュカード」「テキスト→要約」といった単一の変換パスしか持たない。また、Utamemoのような先行サービスは音楽生成に特化しているが、コミュニティ機能が弱く、ユーザーが作ったコンテンツが循環しない構造になっている。

Manashiftは**インプットの柔軟性**・**アウトプットの多様性**・**コミュニティによる知識の循環**を三本柱として設計し、これらの課題を解決する。

### 1.3. ターゲットユーザー

| セグメント | 具体像 | ニーズ |
| :--- | :--- | :--- |
| **中高生・大学受験生** | 定期テスト・センター試験対策 | 楽しく・効率的に暗記したい |
| **語学学習者** | TOEIC・英検・日本語学習者 | 反復に飽きずに続けたい |
| **教員・塾講師** | 授業コンテンツを作りたい先生 | 教材を素早く多形式で作りたい |
| **社会人学習者** | 資格試験・自己啓発 | 通勤時間など隙間時間に学びたい |

## 3. 非機能要件

### 3.1. パフォーマンス

コンテンツ生成はAI処理を含むため、生成完了まで10〜30秒程度かかる場合がある。フロントエンドではローディング状態を適切に表示し、ユーザーが処理中であることを把握できるようにする。

### 3.2. セキュリティ

すべてのAI生成処理はサーバーサイドのtRPCプロシージャ内で実行し、APIキーをクライアントに露出させない。認証が必要な操作（コンテンツ生成・いいね・コメント等）は `protectedProcedure` で保護する。

### 3.3. アクセシビリティ

- モバイルファーストのレスポンシブデザインを採用する。
- キーボードナビゲーションとスクリーンリーダー対応を考慮したshadcn/uiコンポーネントを使用する。

フリーミアムモデルを採用し、学校・教育機関への導入を主軸とした法人プランを設ける。

| プラン | 月額 | 生成クレジット/月 | 対象 |
| :--- | :--- | :--- | :--- |
| **Free** | ¥0 | 10クレジット | 個人・お試し |
| **Starter** | ¥980 | 50クレジット | 個人学習者 |
| **Pro** | ¥2,980 | 200クレジット | ヘビーユーザー |
| **School** | 要相談 | 無制限 | 学校・教育機関 |

各フォーマットのクレジット消費目安：Flashcard=1、Novel=2、Poem=2、Video Script=3、Manga=5（画像生成コストを含む）。
